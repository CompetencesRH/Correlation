# Correlation
Une première approche de la corrélation.
Définition simple :

“Le coefficient de corrélation mesure la force et le sens d’une relation linéaire entre deux variables.”

Palette de valeurs :

environ –0,7 à –1 → corrélation négative forte

–0,3 à 0,3 → faible ou pas de corrélation

0,7 à 1 → corrélation positive forte

Points de vigilance :

La corrélation ≠ causalité.

Elle ne capte que les relations linéaires, pas les formes complexes.
